# Contributing Guidelines

Thank you for considering contributing to this project!  
Please take a moment to read how you can help and how we work together.

---

## Bug Reporting

If you find a bug, please open a new **issue** and include:

- A short, clear description of the problem
- Steps to reproduce it
- Expected vs. actual behavior
- Screenshots or console errors (if possible)
- Your environment info (browser, OS, Node version, etc.)

Check if the issue already exists before creating a new one.

---

## Suggesting New Features

Have an idea? Great!  
Open a **feature request issue** with:

- A clear description of your idea
- Why it would improve the project
- (Optional) Any example or visual mockup
