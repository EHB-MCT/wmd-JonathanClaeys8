# Twitch Chat Monitoring & Moderation System

A real-time Twitch chat monitoring and moderation dashboard with sentiment analysis capabilities. This application connects to multiple Twitch channels, analyzes chat messages for sentiment patterns, and provides moderators with tools to identify and manage suspicious users.

## Usage

### For Moderators

1. **Register/Login**: Create an account or sign in
2. **Add Channels**: Navigate to moderation dashboard and add Twitch channels to monitor
3. **Monitor Messages**: View real-time chat feed with sentiment scores
4. **Review Suspicious Users**: Check the suspicious users list for negative sentiment patterns
5. **Take Action**: Use moderation buttons (Ban, Timeout, Warning, Remove) as needed

### For Viewers

1. **View Analytics**: Access the viewer dashboard for public analytics
2. **Leaderboard**: See top users ranked by activity and sentiment
3. **Charts**: Explore sentiment distribution and activity patterns
