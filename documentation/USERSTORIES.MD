# UserStories

- The user can add more than one channel to track.
- The user can add a channel by typing the name of the channel.
- The user can see all tracket channels in a list, and they are also deletable.
- The user can see the avarage sentimental score of each user with a color and a number
- The user can see the total of messages send from a suspisious user in the suspisious list.
- The user can see how many points are gained or lost in the logs from all channels.
- When the user clicks on a user in the suspisious list, a dropdown opens with 3 buttons: Ban, Time-out, Warning.
